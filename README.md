### ОПЫТ И ПРОЕКТЫ

**Проект: "Videohosting" — Разработка бэкенда для видеоплатформы**  
[Ссылка на GitHub](https://github.com/a23d45/videohosting)

- Разработал полнофункциональный бэкенд для видеохостинга с использованием **Django REST Framework**
- Реализовал **API** для загрузки, просмотра, поиска и редактирования видео, систему оценок и комментариев
- Настроил аутентификацию с помощью **JWT**, фоновую обработку задач через **Celery** и **Redis**
- Развернул проект в **Docker**-контейнерах, обеспечив масштабируемость и простоту деплоя
- *Стек: Python, Django REST Framework, Celery, Redis, PostgreSQL, Docker, JWT*

**Проект: "Gender Classification" — Модель классификации пола на основе FaceNet и SVM**  
[Ссылка на GitHub](https://github.com/a23d45/gender_classification)

- Создал модель машинного обучения для определения пола по фотографии
- Использовал **FaceNet** для извлечения признаков и **SVM** для классификации
- Реализовал пайплайн обработки лиц: обнаружение, выравнивание и преобразование в векторные embedding-и
- *Стек: Python, FaceNet, Scikit-learn (SVM), OpenCV, TensorFlow/Keras*

**Проект: "Logger" — Потокобезопасная библиотека логирования на C++17**  
[Ссылка на GitHub](https://github.com/a23d45/logger)
